# api-testing-playbook

This repository demonstrates a structured approach to API testing using Postman.

## Scope
- Testing REST APIs
- Validating response status codes
- Verifying response data
- Handling negative and edge cases

## APIs Used
- https://jsonplaceholder.typicode.com/users
- https://jsonplaceholder.typicode.com/posts

## Tools
- Postman

## Objective
To practice and demonstrate API testing fundamentals, including request validation, response verification, and identifying potential issues in API behavior.
